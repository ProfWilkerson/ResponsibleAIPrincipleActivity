# ResponsibleAIPrincipleActivity Repo

**Course:** AI Essentials
**Module:** 1
**Type:** Non-graded exploration activity

## Overview
This is a practice/explore activity built for a 100-level Introduction to AI course at a community college. 

A guided, view-only exploration activity that includes:
* 10 scenario cards covering all five responsible AI principles: fairness, transparency, accountability, privacy, and inclusiveness
* A mix of upheld and violated examples.

## File
`index.html` — single self-contained HTML file, no dependencies or external libraries required.

## Instructor Notes
- For each card, students use two dropdowns to select the principle and their judgment, then click Check My Answers for instant feedback (
- Answers are color-coded as fully correct, partially correct, or incorrect.
- A score banner appears at the top of the results, and a collapsible Reference Answer Key at the bottom includes the reasoning for each scenario.
- There's also a Reset button to try again.
- No results are saved. The page resets on refresh.
